# Klant Handleiding

## Bestellen

1. Open `menu.html`.
2. Klik op `Kies` bij een item.
3. Kies formaat, extras en aantal.
4. Open `order.html` om af te ronden.

## Afrekenen

Vul in:

- naam
- telefoon
- ophaaldag en uur
- extra opmerkingen indien nodig

Je kan kiezen tussen:

- betalen bij afhaling
- online betalen (demo)

## Regels

- Afhaling is alleen beschikbaar binnen openingsuren.
- Je kan maximaal 2 dagen vooruit plannen.
- Bij online demo-betaling kom je terug op `order.html` met bevestiging.
